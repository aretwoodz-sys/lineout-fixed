This file is a non‑invasive sync shim. It exposes window.LineoutSync and dispatches/consumes custom events. It will not touch your UI or override applyRows().
How to integrate (custom events) 
// 1) Listen for server rows and render with your own code:
window.addEventListener('lineout:rows', (e) => {
  const { rows, meta } = e.detail;
  // your applyRows(rows) here
});

// 2) Whenever you want to save your current rows:
window.dispatchEvent(new CustomEvent('lineout:save', {
  detail: { rows, meta: { title: 'Checklist', updatedAt: Date.now() } }
}));
